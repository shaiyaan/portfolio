Student Loans

This project analyzes student loan data across U.S. states, focusing on trends in the number of borrowers, total loan balances, and state-level disparities from 2016 to 2021.

View Code Live Demo

The Challenge

Student loans represent a significant financial burden for millions of individuals in the United States, affecting not only borrowers but also the broader economy. With total student loan debt exceeding $1.7 trillion, many Americans struggle to balance repayment with other financial goals, such as saving for retirement or purchasing a home. However, the impact of student loans is not uniform; some states, demographics, and groups face greater challenges than others.

Despite the widespread nature of the issue, clear, accessible, and actionable data is often lacking. Policymakers, researchers, and advocacy groups need processed and visualized insights to understand key trends, identify regions or populations most affected, and guide interventions to reduce the burden of student debt. Without such analysis, it is difficult to address disparities or create targeted, effective solutions.

By analyzing and visualizing trends in student loan data (number of borrowers, total loan amounts, and changes over time), this project aims to fill this gap. The goal is to provide an educational tool for stakeholders, equipping policymakers, researchers, and the general public with the insights necessary to drive informed discussions and impactful decisions about student debt relief and financial education.

Code Examples

def line_chart(state, data, data2):
    avg = []
    for i in range(1, len(data[state])):
        num_borrowers = data[state][i]
        total_amount = data2[state][i]
        if num_borrowers > 0:
            average = total_amount / num_borrowers
            avg.append(average)
        else:
            avg.append(0)        
    return avg

state1_idx = 4  # California
state2_idx = 48  # Wyoming
state1_avg = line_chart(state1_idx, data, data2)
state2_avg = line_chart(state2_idx, data, data2)    

years = ['2016','2017','2018','2019','2020 (June)','2020','2021']

plt.plot(years, state1_avg, label='California')
plt.plot(years, state2_avg, label='Wyoming')
plt.title('Comparison of Average Loan Balances Across Years')
plt.xlabel('Year')
plt.ylabel('Average Loan Balance')
plt.legend()
plt.show()
